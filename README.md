**Example:**
```python
# 1. Two Sum
# https://leetcode.com/problems/two-sum/
# Time: O(n) | Space: O(n)

def twoSum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
```
